# cwApp_Android
CW클랜 공식 앱 개발 일지입니다

# CW 앱은 다음과 같은 기능을 지원합니다
- CW MATCH MAKER를 이용한 CW래더 매칭
- CW래더 전적 확인
- 크리에이터들의 창작물을 모아서 보기
- 마이페이지에서 프로필 이미지 변경
- 클랜 내 공지사항 확인
